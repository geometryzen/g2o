  • new Sprite(board, path, ox?, oy?, cols?, rows?, frameRate?): Sprite

  • Parameters

    • board: IBoard
    • path: string | Texture

      The URL path or Texture to be used as the bitmap data displayed on the sprite.

    • frameRate: number = 0
